A lightweight starter theme built with Foundation and Gulp.

== Description == 
Helium-Fdn is a minimal WordPress starter theme featuring Foundation CSS, Gulp-based build tools, and spacing utilities for rapid development.

== Installation ==
Upload the helium-fdn folder to the /wp-content/themes/ directory.
Activate the theme through the 'Appearance > Themes' menu in WordPress.
Run npm install and gulp in the theme directory for development.
